The Mining Industry

context

Mining is something done for a really long period and time and new inventions had made the process easier. There are different types of tools for mining such as softwares, hardwares for land, underground, or air use.

Current Software

Software

There are a lot of software used in the mining industry. Their purpose are to track machines, progress, and profits as well as training workers.

Three commonly used software are:

  • Sap
  • Datamine
  • Micromine

The first software is sap

The software optimizes efficiency and reduces downtime without damaging the machines. It allows companies to keep track of sales, shipping, profits etc. The software currently can be run on linux and windows

The Second software is Micromine

The software helps you manage and see mining and exploration data with predictions and graphs It also records data from equipment, employees and materials. It stats their status and allows for greater control. It only supports Windows 10 pro 64 bit

The third software is Datamine

, It provides consultors who gives advice , It also provides training to understand the technical things about mining and business solutions to improve a project ,It only supports windows 8.1 and up

Current Hardware

Hardware

There are a lot of hardware used in the mining industry. Their purpose are to help with the process of scouting an area, mining, and during emergencies.

Three commonly used hardware are:

  • Drones
  • Refuge Chambers
  • Blasthole Drills

One commonly used in the industry are drones

  • can be used to survey an area and plan for an operation
  • inspects the underground, and helps find problems.
  • It can map out the surface an area in different ways such as show elevations

Another commonly used in the industry are Refuge chambers

  • Refuge chambers are built in case a mine collapses.
  • It houses workers inside until rescue comes.
  • It is usually stored with the basics for survival and oxygen

A third commonly used in the industry are blasthole drills

  • Most basic machine used to drill into the ground to place explosives
